# Simple_Slot_Game
- A feature-rich slot machine game with realistic animations, age verification, and immersive win/lose effects.
**Live Demo:** [https://redluzon.github.io/Simple_Slot_Game/SLOT-GAME/HTML-FILE/slot.html]
]

### Features
- Age verification modal (18+ gambling content warning)
- Individual reel spinning with 3 fruit symbols (🍎 Apple, 🍇 Grape, 🍊 Orange)
- Jackpot celebration with confetti effects and glowing reels
- Loss animation with screen shake and red flash overlay
- Smooth spin animations with 3D rotation effects
- Responsive design for mobile, tablet, and desktop
- Landscape mode support
- Home screen with floating animated icons

### Tech Stack
- HTML5, CSS3, JavaScript
- CSS Grid & Flexbox layouts
- CSS keyframe animations (spin, shake, pulse, bounce)
- Dynamic DOM manipulation

### Game Rules
- Match all 3 reels with the same symbol to win JACKPOT!
- Spin individually or reset all reels
- Two-symbol matches trigger loss animation
- Premium casino-themed UI with gradient backgrounds

### Pages Included
- Landing Page with logo, subtitle, and Play button
- Age Verification Modal for legal compliance
- Main Game Arena with 3 interactive slot reels
- Jackpot Celebration Overlay with confetti
- Loss Screen with try-again prompt
